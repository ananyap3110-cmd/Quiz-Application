# Quiz Application

A simple Quiz Application developed using Python and Jupyter Notebook.  
This project allows users to answer quiz questions, calculate scores, and store quiz-related data using JSON files.

## Features
- Multiple-choice quiz questions
- Score calculation
- Question management using JSON
- Beginner-friendly project structure
- Interactive notebook interface
- Simple and easy-to-understand code

## Technologies Used
- Python
- Jupyter Notebook
- JSON

## Project Files

| File Name | Description |
|---|---|
| `normal.ipynb` | Main quiz application notebook |
| `extra.ipynb` | Additional quiz features |
| `tkinter.ipynb` | GUI-based quiz implementation using Tkinter |
| `questions.json` | Stores quiz questions |
| `scores.json` | Stores user scores |

## How to Run the Project

1. Install Python
2. Install Jupyter Notebook
3. Open the project folder
4. Run the notebook files using Jupyter Notebook
